  • Can load with selections (demoed below). Useful when 'Viewing' a pre-existing record
  • Mobile-friendly
  • Searchable (with <prime-expanding-search>)

Enrollment List

<prime-enrollment-list [rowItems]="enrollmentListDummyData"></prime-enrollment-list>
  • Includes search (only searches bolded part of row).
  • Rows are expandable
  • Animations for row expansion and search
  • Known Bug: The View Dropdown and Search don't play well together currently.
  • View Dropdown doesn't properly filter top level rows.

Expanding Search

</prime-expanding-search>

  • Doesn't handle filtering/search, use (onChange) and handle elsewhere.
